# Bloodware, Inc. – Innovation & Collaboration Hub

**Formerly Blood Gang, Inc.**

Bloodware, Inc. is the evolution of Blood Gang, Inc.—a collaborative, innovation-driven hub pushing boundaries in gaming, software, and hardware. After years of building and creating under the Blood Gang brand, the company rebranded to unify all projects under one professional identity.

## Core Technologies

- [Lua](https://www.lua.org/) – Game scripting and automation  
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) – Web apps, APIs, and tools  
- [Python](https://www.python.org/) – Automation, backend services, and data handling  
- [C++](https://isocpp.org/) – Performance-critical software and game engines  
- [Unreal Engine](https://www.unrealengine.com/) – High-fidelity, immersive horror and shooter games
